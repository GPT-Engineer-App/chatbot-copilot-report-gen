## Setup

```sh
git clone https://github.com/GPT-Engineer-App/chatbot-copilot-report-gen.git
cd chatbot-copilot-report-gen
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
